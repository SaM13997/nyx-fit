# Features Suggestions for Weights

## Core Functionality
- [X] **Log Weight**: Simple input to log current weight with date and optional time.
- [X] **Weight History**: List of past weight entries (Date, Weight, +/- change).
- [X] **Edit/Delete Entries**: Ability to correct mistakes.
- [X] **Notes**: Add notes to weight entries (e.g., "Post-cheat meal", "Morning empty stomach").

## Visualization & Analytics
- [X] **Weight Trend Chart**: Line graph showing weight over time (1W, 1M, 6M, 1Y, All).
- [X] **Trend Line**: Moving average to smooth out daily fluctuations.
- [X] **BMI Calculator**: Auto-calculate BMI based on height (needs profile height) and current weight.
- [X] **Progress Stats**: Total weight lost/gained, average weekly change.

## Goals & Targets
- [X] **Target Weight**: Set a goal weight.
- [X] **Milestones**: "5lbs lost", "Halfway to goal" badges or notifications.
- [X] **Weekly Goal**: Target rate of loss/gain (e.g., -1lb/week).

## Advanced Integrations (premium only)
- [X] **Photos**: Attach progress photos to weight entries.
- [ ] **Body Fat %**: Log body fat percentage if available.
- [ ] **Export/Import**: CSV export of data.
